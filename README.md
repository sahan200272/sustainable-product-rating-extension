# 🌱 Sustainable Product Rating Extension

A MERN-based web application that acts as a sustainable product rating extension for online shopping platforms. The system evaluates products based on environmental and ethical criteria to promote responsible consumption and support **UN Sustainable Development Goal 12 – Responsible Consumption and Production**.

---

# 📌 Project Overview

The Sustainable Product Rating Extension enables users to:

- Add sustainable products
- Update product sustainability data
- Retrieve sustainability ratings
- Delete products
- Analyze sustainability metrics

The system evaluates products using sustainability indicators such as:

- Environmental impact
- Ethical sourcing
- Carbon footprint
- Packaging sustainability
